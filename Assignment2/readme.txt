Web Design and User Experience Engineering

Assignment 2 - Portfolio


It is my Portfolio where users can view my professional details online.
The site lands on the Home page, which previews my professional details and features a navigation bar on the left side of screen.
- HOME : It loads the top of the page.
- ABOUT : It takes the page to the about division of portfolio.
- SERVICES : It takes the page to the services division of portfolio.
- CONTACT : It opens a window from default mail app to send mail.

Tags used - 
~ Favicon - It adds an image/icon to the title of the page in the browser tab.
~ <header> - It works as container for the introduction of page. I used it to preview the page name.
~ <footer> - It defines the footer section of the HTML page. I have used it to preview full name, provide phone number and email.
~ <nav> - It defines a section for navigation links. I have used it to navigate to different divisions on the page, i.e. Home, About, Services and Contact.
~ <style> - It is used to define the required css of the elements in the HTML page.
~ <img> - It is used to link an image to the HTML page. I have used .gif and .jpg.
~ <a> - Hyperlink - Used to define a hyperlink. the 'href' attribute in <a> is used to indicate the link's destination. I have used it,
    - <nav> - in the navbar to link all 3 divisons.
    - to link to Linkedin.
    - to open Resume document.
    - to go to Certifications divisions.
~ <br> - It is used to break the line and move to the next line.
~ <figure> - To create photo gallery for certifications. <figcaption> used to create caption for each figure.

~ style.css - External Style Sheet created which includes Media Queries.
~ Responsiveness - @media - It uses the @media rule to include a block of CSS properties only if a certain condition is true.
    - for iPad - 768px
    - for Mobile - 350px to 365 px
~overflow - Scroll -specifies whether to clip the content or to add scrollbars. Used in testimonials division.
~ tel - to call the phone number from the page directly.
~ mailto - to open default mail app to send email.
